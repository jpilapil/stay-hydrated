## Stay Hydrated

Drinking water is important, this lightweight desktop application is here to remind you.


### Technologies used

- Electron.js\
- React.js\
- Moment.js\
- Bootstrap


### Installation

- Clone this repository:
<pre><code>git clone https://github.com/jpilapil/stay-hydrated.git
</code></pre>

- Run:
<pre><code>npm start
</code></pre>
or
<pre><code>yarn start
</code></pre>
